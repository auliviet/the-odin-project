# Admin Dashboard

[The Oding Project](https://www.theodinproject.com): Admin Dashboard.
Exercise as part of the Full-stack JavaScript curriculum on working with CSS Grids.

## Preview link

[Github Pages](https://auliviet.github.io/the-odin-project/admin-dashboard/)

## Project Overview

The admin dashboard project creates a mock page for users to see an hypthetical list of projects, annoucement and team members from a company. The data is static.
