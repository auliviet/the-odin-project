# Rock Paper Scissors

[The Oding Project](https://www.theodinproject.com): Rock Paper Scissors.
Exercise as part of the Foundations curriculum on working with JavaScript functions.

## Preview link

[Github Pages](https://auliviet.github.io/the-odin-project/rock-paper-scissors/)

## Instructions

The full set of instructions can be found at [The Odin Project ](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors).

- Create function getComputerChoice that will randomly return either ‘Rock’, ‘Paper’ or ‘Scissors’.
- Get playerSelection from the user
- Write a function playRound that plays a single round of Rock Paper Scissors
- Create a function playGame to play 5 rounds, keep score and report the winner
