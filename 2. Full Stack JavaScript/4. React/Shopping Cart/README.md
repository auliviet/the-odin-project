# MYSTORE

## About The Project

[The Oding Project](https://www.theodinproject.com): Shopping Cart.
Exercise as part of the Full-stack JavaScript curriculum on React.

## Preview link

[Github Pages](https://auliviet.github.io/the-odin-project/shopping-cart/)

## Project Overview

Ecommerce application with client-side routing. It is built with React, React-Router, TypeScript and Vite for the build. Unit Tests using Vitest and React-Test-Library.
The goal of this exercise is to create a multi-pages site with React, Loaders and Actions in React Router. Product data is fetched from [FakeStore API](https://fakestoreapi.com/).

The project offers the below functionalities:

- Dynamic cart using OutletContext.
- Get product data from the fetch() API.

## Instructions

The full set of instructions can be found at [The Odin Project ](https://www.theodinproject.com/lessons/node-path-react-new-shopping-cart).

- Create a new React project.
- Structure your application into at least three pages:
  - Homepage
  - Product page
  - Cart page
- Display an input field on the product page, which lets a user manually type in how many items they want to buy.
- Indicate how many items are in your cart in the navbar.
- Display items and their quantities in the cart page.
